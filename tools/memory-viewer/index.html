How to read the 3D view

Each dot is one captured moment. Hue encodes activity (walking, working, …). Brighter dots = higher scene confidence. Vertical depth (Z) follows time; the teal polyline connects the filtered events in chronological order. Red/green/blue sticks at the origin are XYZ axes (not events).
